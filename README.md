## コマンド

### 1. プロジェクトのクローン
```bash
git clone https://github.com/NAoMi-1242/japanese_c_lang.git
````

### 2. ディレクトリの移動
```bash
cd japanese_c_lang
```

### 3. Dockerイメージのビルド
```bash
docker-compose build
```

### 4. 開発コンテナの起動と接続
```bash
docker-compose run --rm dev
```
